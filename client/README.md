# Todo App

## Installation

Intall Todo App:

```
git clone https://github.com/Timo-Turtiainen/todo-app.git
cd todo-app
npm install
npm run start
```

### Usage

Create your own .env file

```
MONGODB_URI = <your mongo database url with username & password>
PORT= <server runs on port>
SECRET= <your own password hash>
```
